You need to activate the venv_crypto for the program to work.
(Use Python3)

To activate it (in Linux):
1) Navigate to the project directory in Terminal.
2) Run 'source venv_crypto/bin/activate'.
3) Now you can run the program with the proper libraries 'python3 main.py'.
4) Run 'deactivate' to deactivate the venv

If you are using Windows:
- Run it in WSL, repeatig the steps above.
OR
- Ensure that you have the necessary libraries installed:
